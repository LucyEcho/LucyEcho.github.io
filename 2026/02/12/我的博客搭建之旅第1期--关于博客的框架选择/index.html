关于博客的框架选择

动态博客

动态博客，主要特点是可以在线编辑，最有名的就数Wordpress框架了，其他就是Typecho，Wix，Halo等，占用资源大，部署较为繁琐，维护相对困难，一般不推荐去搭建博客，最多是用于电商的独立站和一些企业官网。

静态博客

静态博客，就是我们今天的重点了，一般是通过生成工具去编译我们所撰写的博客，最后生成由html，css，js等静态文件组成的项目，然后我们再将项目托管到Github，Cloudfare，Vercel等平台上去部署访问；常用的生成工具有：Hexo，Jekyll，Hugo 等

在线平台

在线平台，我主要用过两款Notion，Obsidian，都还不错。

我的选择

Hexo+Github+免费域名+Cloudfare加速

我的博客搭建之旅第2期--Hexo的环境搭建
Hexo的环境搭建一、前置准备 首先准备一个 github 账号（没有的可以先去注册一个），用于托管博客代码； 然后要下个git 用于将博客代码部署到GitHub上去； 还要下载node.js，我们要用到npm工具去下载Hexo； 二、GitHub连接 首先新建文件夹blogs,进入并打开cmd命令行输入bash，进入git bash，然后依次输入下列命令： 123git config --global user.name "Your Name"git config --global user.email "youremail@example.com"ssh-keygen -t rsa -C "youremail@example.com" #看到要输入什么可以直接回车！ 然后打开C:\Users\Administrator\.ssh\id_rsa.pub文件，全选复制； 打开GitHub找到Setting–>SSH and GPG keys，添加SSH key，复制进去； 这样收到邮箱就连接成功了 三、...
